✦ Input Guide

How to structure your input file

Use these formatting rules in your .docx file so our tool can parse it perfectly.

H1 — Paper Title

Title Style

Your paper title should use Word's Heading 1 style or be the first line of the document in bold, 14–16pt.

Heading 1 → Deep Learning Approaches for Fault Detection in Power Grid Systems
H2 — Sections

Section Headings

Use Word's Heading 2 style for all major IEEE sections like Abstract, Introduction, Methodology, Results, etc.

Heading 2 → I. Introduction II. Related Work III. Methodology IV. Results
Body Text

Paragraph / Body

Keep body text as Normal style in Word (12pt Times New Roman recommended). Avoid custom styles for regular paragraphs.

Normal style, 12pt The proposed method achieves a 94.3% classification accuracy across 5 benchmark datasets…
References

References Section

Add a Heading 2 labeled "References" and list them in IEEE citation format — numbered, bracketed.

Heading 2: References [1] A. Smith, "Title of paper," in Proc. IEEE Conf., 2023, pp. 1–5. [2] B. Jones et al., "Another paper," IEEE Trans., vol. 12…
💡 Pro Tips

Things to double-check before uploading

Abstract: Place it right after the title as a separate paragraph with the label "Abstract" bolded or as Heading 2.  •  Figures: Embed them inline with captions directly below (e.g., "Fig. 1. System Architecture").  •  Tables: Caption goes above the table in IEEE style (e.g., "TABLE I — Comparison of Methods").  •  Equations: Use Word's equation editor — they'll be preserved.  •  Avoid: Text boxes, custom fonts, multi-column manual layouts (we handle column formatting for you).
